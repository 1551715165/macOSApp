# M-Renamer
M-Renamer(Method-Name-Renamer)类方法名修改器，可以解析整个项目大多数类的方法，一键批量修改，引用处自动修改，效率高，省时省力，修改后的方法名规范，提高过审机率，支持Swift(未来支持)、Objecive-C双语言。

[查看详情](./M-Renamer)
